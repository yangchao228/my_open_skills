# Work Skills

[English](README.md)

`skills/work/` 存放可复用的结构化工作产出 skills。

## Ready Skills

- [resume-interview-generator](resume-interview-generator)：根据简历和岗位生成结构化面试题、追问路径、评分标准和风险验证点。
- [project-weekly-report](project-weekly-report)：基于本地仓库历史、标签、文档和产物生成有证据链的项目周报。
- [create-plan](create-plan)：在代码、文档、仓库或工作流变更前生成简洁实施计划。
- [doc-coauthoring](doc-coauthoring)：引导重要文档完成上下文对齐、分节起草、读者测试和最终检查。
