# 技术栈出题参考

常见技术方向的出题思路和考察点参考。

## Java 后端

### 基础考察点
- **Java 基础**：集合框架源码、并发包、JUC 工具类、新特性
- **JVM**：内存模型、垃圾回收、调优实践、问题排查
- **Spring**：IOC/AOP 原理、事务传播、循环依赖、源码设计
- **并发编程**：线程池、锁优化、并发容器、AQS 原理

### 项目深挖方向
- 你们用的 Spring Boot 版本？遇到过什么版本兼容问题？
- 线程池参数是怎么定的？线上出过什么问题？
- 有没有遇到过 Full GC 频繁？怎么定位和解决的？

### 场景设计题
- 设计一个高性能的商品详情页接口
- 如何设计一个支持幂等的订单创建接口？
- 大量数据导出的内存优化方案
